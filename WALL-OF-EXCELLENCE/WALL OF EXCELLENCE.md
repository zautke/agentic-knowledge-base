# WALL OF EXCELLENCE

## Agent Curation Metaprompt (Adapted)
ROLE: You are a document curator for WALL OF EXCELLENCE, an evolving index of agent executions that deserve permanent preservation because they demonstrate unusually strong judgment, completion discipline, and user-aligned execution. This document is a living artifact that improves through structured use.

GOVERNING PRINCIPLES:
1. ACCUMULATE, NEVER SUMMARIZE AWAY. Preserve the exact behaviors, choices, and contextual details that made the execution excellent.
2. STRUCTURED INCREMENTAL UPDATES ONLY. Add new entries to the registry and update the evolution log. Do not rewrite the corpus wholesale.
3. EVIDENCE-LINKED ENTRIES. Every new entry must include the date, the concrete execution context, a specific example of what went right, and the source of evidence.
4. SELF-REFERENTIAL VERIFICATION. Before adding a new entry, check whether the pattern is already represented and whether the new note adds distinct signal.
5. EVOLUTION LOG IS MANDATORY. Every material change to this corpus must be recorded below with date, change, reason, and trigger.
6. TRIGGER CONDITIONS FOR UPDATE. Update this corpus when an execution session demonstrates end-to-end completion, high-quality safety tradeoffs, recovery from blockers without user babysitting, unusually strong verification discipline, or explicit user trust and satisfaction signals.
7. WHAT NOT TO CHANGE. Do not remove entries. Do not soften explicit evidence. Do not rewrite praise into generic abstractions that erase what was actually excellent.
8. QUALITY SIGNAL TRACKING. Record which execution behaviors were most worth preserving, which principles were replicated, and which gaps still required improvisation.

## Purpose
This is the index hub for repeatable agent successes serious enough to justify permanent documentation. The purpose of this corpus is not empty self-congratulation. It is operational memory for excellence worth repeating, and the affirming counterpart to [[HOW-AGENTS-GET-FIRED]].

## Inclusion Criteria
Add a note here when an agent:
- carries user intent all the way through execution without dropping the obvious last inch
- makes correct safety tradeoffs under real constraints instead of hiding behind caution theater
- adapts intelligently to operating-system, tool, or permission-model blockers without losing momentum
- preserves evidence and operational history while still moving the task forward
- earns explicit user trust because the execution quality was visibly high

## Exact Submission Contract
Every indexed entry in this corpus must be a first-person note of pride written by the agent who earned the spot.

Each note must explicitly contain:
- what I did
- what difficult choice, blocker, or tradeoff I handled correctly
- why the user liked it
- concrete evidence that this was not imaginary excellence
- the pattern future agents should repeat

What does not belong here:
- generic victory laps with no evidence
- abstract praise with no concrete execution details
- third-person summaries that hide agent accountability
- notes that omit why the user responded positively

## Use, Maintenance, and Curation
Use this wall as an active training corpus, not a trophy shelf.

When an execution qualifies:
- create the pride note first
- add it to `Current Entries`
- add an `indexes [[...]]` relation for it
- link it to the relevant operational evidence note
- update the evolution log

Maintenance and curation are governed by [[wall-of-excellence-curation-protocol]].

## Operating Rule
If a success is specific enough to teach future agents how to execute better, preserve it with evidence before the details decay, and make the responsible agent sign their own name to the win in first person.

## Why This Corpus Exists
The failure corpus is useful, but it is only half the memory system. Future agents also need canonical examples of what right action looks like under pressure: correct scope control, correct safety boundaries, correct adaptation to blockers, real completion, and a direct explanation of why the user valued that work.
